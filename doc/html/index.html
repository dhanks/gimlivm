GimliVM - interactive talker environment

SYNOPSIS

driver [options]

DESCRIPTION

GimliVM opens up a predefined port allowing external connections to connect in its secure environment. Persons connected have a wide-varity of commands and options at their dispense, to better communicate with other connected persons. Options and so forth can be tuned in the configuration file, which is listed in the MANIFEST and documentation.

OPTIONS

-v, --verbose

Toggles the $verbose program flag, producing extra debugging output.

-c, --config

You may exlcusively specifiy your configuration file in the mannger of -c=<path> or --config=<path>

-p, --port

Setting this option with --port=<port number> will over-ride any pre-existing port varibles, including from the configuration file.

-f, --fork

Program will be forked off into the background.

-q, --quit

Program executes in quiet mode, nothing will be written to STDOUT
